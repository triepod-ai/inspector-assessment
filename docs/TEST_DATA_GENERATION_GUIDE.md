## Overview

The TestDataGenerator provides intelligent, context-aware test data generation for MCP tool assessment. For detailed documentation, please see the split guides above.

**Key Features:**

- **Field Type Handlers** - Smart detection of URL, email, path, date, and 20+ other field types
- **Boundary Testing** - Automatic generation of min/max values based on schema constraints
- **Tool Category Awareness** - Calculator, search, system, URL fetcher tool patterns
- **Scenario Generation** - Happy path, edge case, boundary, and error scenarios
